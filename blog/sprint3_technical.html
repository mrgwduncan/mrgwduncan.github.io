Explain control flow and loops

Control flow is the order that a program runs code and execution contexts are created, which is determined by the lexical environment (written order and location of code). For, while and do while loops affect the control flow because the while looping through no other action may be taken as javascript is synchronous.

To relate this to the car analogy above Function accelerate (speedLimit)
If (ignition === on ) {
Let speed = 0
While (speed < speedLimit)
speed ++
}

If the car is turned on, accelerate to the speed limit, however this example is not the best as the car can still do other actions while accelerating, like turn or brake cars are asynchronous.

Explain the difference between accessing data from arrays and objects An array is an ordered list of values, they can be primitives, objects and functions or even another array separated by commas.

Let's keep it simple with a list of cars
brandsOfCars=[”bmw”, ”tesla”, “toyota”, “ford”, “holden”]

You can refer to items in the list by putting the key value between square brackets, the list starts at 0 so bmw would be

brandOfCars[0]

There are built in methods for arrays for example length which will tell you how many items in the array.

brandOfCars.length would return 5
Object are a little different however they are a list of key value pairs which are referred to by the key rather than the index (the position in the list)

Let Car = {
“Brand”: “Tesla”
“model”:”Roadster”
“Wheels”: 4
“Price”: 200000
}

So we can now get the brand of the car by using car.brand or the model by car.model


So what’s the difference, and when do we use these different data storage techniques?

Objects are used to store things with properties where arrays are used to store list of data in a single variable.
